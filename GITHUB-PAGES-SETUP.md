## 启用 GitHub Pages（2 分钟）

### 步骤

1. **打开** https://github.com/cittaverse/brain-music/settings/pages

2. **Build and deployment**
   - Source: Deploy from a branch
   - Branch: 选择 `main` 
   - Folder: 选择 `/ (root)`
   - 点击 **Save**

3. **等待部署**（约 1-2 分钟）

4. **获得公开链接**
   - 格式：`https://cittaverse.github.io/brain-music/`
   - 页面会显示 "Your site is live at..."

## 自定义域名（可选）

如果想用自定义域名（如 `brain-music.cittaverse.com`）：

1. Settings → Pages → Custom domain
2. 输入你的域名
3. 配置 DNS CNAME 记录
4. 等待 DNS 生效（约 10 分钟）
