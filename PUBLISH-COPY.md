```
**项目背景**

读《Musicophilia》（音乐癖）时受到启发：
- 音乐真的能改变大脑
- 音乐治疗在临床已有应用
- 但普通人没有工具自我评估

**产品定位**

Brain Music - 音乐认知训练系统

不是播放器，是训练器。

**核心功能**

1. 6 训练目标：专注/情绪/创意/活力/记忆/睡眠
2. Spotify 歌单播放
3. 6 题感知打卡量表（身体/情绪/记忆/思维/重复/呼吸）
4. 8 脑区热力图可视化（额叶/顶叶/枕叶/颞叶/小脑/基底节/边缘系统/脑干）

**技术实现**

- 前端：原生 JS + HTML + CSS
- 存储：localStorage（本地持久化）
- 算法：6 问→8 脑区映射，分数归一化 0-5 分
- 部署：Netlify Drop（30 秒）

**脑区映射算法**

6 个问题 → 8 个脑区分数：
- 身体感受 → 小脑/脑干
- 情绪 → 边缘系统
- 记忆 → 颞叶
- 思维 → 额叶
- 重复感 → 基底节
- 呼吸 → 顶叶/枕叶
```
